Private-Label Coffee Marketing Analytics in R

Overview

This project uses R to support a private-label coffee launch strategy for a grocery retail setting.

It combines customer segmentation, product preference analysis, and visual dashboard reporting to identify target customers and recommend a product positioning strategy.

Tools Used

* R
* RFM Analysis
* K-Means Clustering
* Conjoint Analysis
* PCA
* Data Visualization

What This Project Shows

* Customer segmentation
* Retail transaction analysis
* Product preference analysis
* Private-label product strategy
* Dashboard-style business reporting
* Marketing analytics in R

Dashboard

A dashboard PDF is included in the repository to summarize the key findings and product recommendation.

How to Run

Open and run:

private_label_coffee_marketing_analytics.R
